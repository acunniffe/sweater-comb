# @snyk/sweater-comb

OpenAPI linting rules for Snyk APIs.

# Usage

## NPM

`npm install @snyk/sweater-comb` in your project.

Then configure Spectral in your project to extend these rules.

Example `.spectral.yaml`:

```
extends:
  - "@snyk/sweater-comb"
```
